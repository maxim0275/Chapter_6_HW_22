Задание 1


Создан новый контроллер и шаблон для отображения страницы с подробной информацией о товаре. На этой странице показана вся информация о товаре.

Задание 2


Добавлен в шаблон главной страницы код для отображения списка товаров с помощью цикла. Чтобы карточки товаров выглядели одинаково, обрезано отображаемое описание до первых 100 символов.

Задание 3


Выделен общий (базовый) шаблон, который включает общие элементы страницы, такие как шапка, подвал и стили. Создан подшаблон для главного меню, который можно включать в другие шаблоны.
