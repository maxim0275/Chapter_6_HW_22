Задание 1
    - создано новое приложение для работы с пользователями в проекте Django.
    - определена модель пользователя с использованием AbstractUser, с заданием электронной почты как поле для авторизации.
    - добавлены дополнительные поля в модель пользователя: аватар (изображение), номер телефона, страна.

Задание 2
В сервисе реализована функция аутентификации, а именно:
    - регистрация пользователя:
        - создана форма для регистрации пользователя с полями электронной почты и пароля.
        - добавлена логика регистрации в представлении, пользователю позволяется создать учетную запись.
        - реализована отправка приветственного письма пользователю после регистрации (протестирована отправка писем в Django без использования реальной почты с помощью консольного почтового бэкэнда. Выводиться письмо в консоль, и  видно, что оно "отправлено".).
    - авторизация пользователя:
        - настроена форма и представление для авторизации пользователя по электронной почте и паролю.

Задание 3
    - закрыт для анонимных пользователей доступ к страницам, связанным с управлением продуктами. Только авторизованные пользователи могут просматривать, создавать, изменять и удалять продукты. Общедоступной остаталась только страница просмотра списка товаров.
