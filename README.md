Задание 1. Настройка проекта
    - создана новая директория DjangoProject для проекта и настроено виртуальное окружение poetry;
    - инициализирован новый проект Django внутри директории DjangoProject.

Задание 2. Создание и настройка приложения
    - создано новое приложение под названием catalog в проекте;
    - зарегистрировано приложение в настройках проекта;
    - настроена маршрутизация для приложения с помощью добавления соответствующих URL-адресов.

Задание 3. Создание шаблонов
    - подготовлено два HTML-шаблона: для домашней страницы home.html и для страницы с контактной информацией contacts.html;
    - для стилизации страниц использован Bootstrap.

Задание 4. Реализация контроллеров
    - создан контроллер home_view для отображения домашней страницы;
    - создан контроллер contacts_view для отображения страницы с контактной информацией;
    - настроена маршрутизация для созданных контроллеров.

после запуска сервера командой "python manage.py runserver"
страницы доступны по адресам: http://127.0.0.1:8000/contacts/, http://127.0.0.1:8000/home/
